---

## ðŸš© Problems / Issues

While **Sitefinity** offers out-of-the-box (OOTB) reset password screens, there are limitations. You can find more details about the **OOTB reset password** functionality [here](https://hawjeh.com/blog/sitefinity-ootb-reset-password). 

For more customization, **we need to build a custom widget** and content type to support the password reset process.

All code snippets used in this solution are available in my **[GitHub Gist](https://gist.github.com/hawjeh/c8daf9367fa84c4777cca341d6750a42)**.

## ðŸ”¥ Solution Overview

Here's a step-by-step guide to building a custom reset password solution:


### 1. Modify and Place the `login.html` File

- **Modify the `login.html` file** and place it in the Sitefinity folder: **Content/Pages/login.html**.

  You can find the file content [here](https://gist.github.com/hawjeh/c8daf9367fa84c4777cca341d6750a42#file-content_pages_login-html).


### 2. Create a Custom Module to Store Request and User IDs

- **Create a custom module** to hold temporary **request ID** and **user ID** for password reset verification.

  ![screen1](/assets/blogs/sitefinity-custom-reset-password/screen1.png)

- **Content Type Structure**:

  ![screen2](/assets/blogs/sitefinity-custom-reset-password/screen2.png)

- **Hide the Content Page from Navigation** in the backend.

  ![screen3](/assets/blogs/sitefinity-custom-reset-password/screen3.png)

### 3. Configure SMTP Profile

- **Configure SMTP Profile** to ensure email notifications are sent out for password reset.

  ![screen4](/assets/blogs/sitefinity-custom-reset-password/screen4.png)

### 4. Configure Advanced Settings

- In **Security > Membership Providers > Default > Parameters**, set:

  - **enablePasswordRetrieval** to **true**

    ![screen5](/assets/blogs/sitefinity-custom-reset-password/screen5.png)

  - **enablePasswordReset** to **true**

    ![screen6](/assets/blogs/sitefinity-custom-reset-password/screen6.png)

### 5. Add Custom Code to `Global.asax.cs`

- Add the necessary custom code to **`Global.asax.cs`**.

  Find the code in my **[GitHub Gist](https://gist.github.com/hawjeh/c8daf9367fa84c4777cca341d6750a42#file-global-asax-cs)**.

### 6. Set Up a Forgot Password Page

- Create a **Forgot Password Page** in the **Pages** section of Sitefinity.

  ![screen7](/assets/blogs/sitefinity-custom-reset-password/screen7.png)

- Choose a **default MVC page** for this page.

  ![screen8](/assets/blogs/sitefinity-custom-reset-password/screen8.png)

- Install the **ForgotPassword** widget.

  ![screen9](/assets/blogs/sitefinity-custom-reset-password/screen9.png)

### 7. Reset Password Screen

- After setting everything up, test the **reset password screen**:

  ![screen10](/assets/blogs/sitefinity-custom-reset-password/screen10.png)  
  ![screen11](/assets/blogs/sitefinity-custom-reset-password/screen11.png)

### 8. Receive the Password Reset Email

- Once the user requests the password reset, they will receive an email with a reset link.

  ![screen12](/assets/blogs/sitefinity-custom-reset-password/screen12.png)

### 9. Route to Change Password Screen

- The email link will route the user to the **change password** screen.  

  ![screen13](/assets/blogs/sitefinity-custom-reset-password/screen13.png)  
  ![screen14](/assets/blogs/sitefinity-custom-reset-password/screen14.png)

### 10. Test the Login After Reset

- After changing the password, the user will be able to **log in** with their new credentials.  

  ![screen15](/assets/blogs/sitefinity-custom-reset-password/screen15.png)
