# üîê Azure AD Integration with Sitefinity (OpenID Connect)

Enable seamless **SSO (Single Sign-On)** with Microsoft Azure AD using OpenID Connect in your Sitefinity instance.

---

## ‚òÅÔ∏è Azure AD Setup

1. Go to **Azure Active Directory** ‚Üí **App registrations** ‚Üí **New registration**.
2. Enter a name for your application (you can skip Redirect URI for now), then click **Register**.
3. Navigate to the **Authentication** tab ‚Üí **Add a platform** ‚Üí choose **Web**.
4. Under Redirect URIs, enter: ```http://<your-sitefinity-instance>/Sitefinity/Authenticate/OpenID/signin-custom```
5. Enable **ID tokens** under ‚ÄúAuthorization endpoint‚Äù.


## üèóÔ∏è Sitefinity Setup

2. Go to the backend ‚Üí **Administration** ‚Üí **Advanced** ‚Üí  
**Authentication** ‚Üí **SecurityTokenService** ‚Üí  
**AuthenticationProviders** ‚Üí **OpenIDConnect**.
3. Apply the following configuration:

## üîß Optional Enhancements

1. Create `/Content/Pages/login.html`
