## ‚òÅÔ∏è Setting Up Azure OpenAI

Follow these steps to configure Azure OpenAI for integration with Sitefinity:

### 1. Access Azure Portal

Go to the [Azure Portal](https://portal.azure.com/) and log in with your Azure credentials.

### 2. Create a Resource Group

Create a new **Resource Group** within your Azure Subscription to organize your OpenAI resources.

### 3. Create an OpenAI Service

Within the resource group, create a new **Azure OpenAI** service.

![screen1](/assets/blogs/sitefinity-open-ai/screen1.png)

### 4. Search and Select OpenAI

In the **Azure Marketplace**, search for **"OpenAI"**, select **Azure OpenAI**, and create the service.

![screen2](/assets/blogs/sitefinity-open-ai/screen2.png)

### 5. Obtain API Keys and Endpoint

Navigate to **Keys and Endpoints** in the Azure OpenAI service. Save either **KEY 1** or **KEY 2** and the **Endpoint** for later use.

![screen3](/assets/blogs/sitefinity-open-ai/screen3.png)

### 6. Access Azure OpenAI Studio

In the **Overview** section, click **Go to Azure OpenAI Studio** to start configuring your OpenAI models.

![screen4](/assets/blogs/sitefinity-open-ai/screen4.png)

### 7. Create a Deployment

In the Azure OpenAI Studio, create a **new Deployment** and select the **gpt-35-turbo** model. Set a **Deployment Name**.

![screen5](/assets/blogs/sitefinity-open-ai/screen5.png)

### 8. Save the Deployment Name

Remember the **Deployment Name** as it will be required for Sitefinity configuration.

## üèóÔ∏è Configuring Sitefinity

### 1. Install the AIServiceConnector NuGet Package

In your **Sitefinity project**, install the **Progress.Sitefinity.AIServiceConnector** NuGet package to enable AI service integration.

### 2. Log in to Sitefinity CMS Dashboard

Log into your **Sitefinity CMS Dashboard**, and navigate to **Administration ‚Üí Modules & Services**.

### 3. Install AI Services

Install the **AI Services** module in Sitefinity to enable AI functionalities.

![screen6](/assets/blogs/sitefinity-open-ai/screen6.png)

### 4. Configure AI Services

Go to **Administration ‚Üí Settings ‚Üí AI Services** to configure the API keys and endpoints:

- **API Key**: Use either **KEY 1** or **KEY 2** from Azure (Step 5 above).
- **Endpoint**: Use the **Endpoint** provided by Azure.
- **Deployment Name**: Enter the **Deployment Name** from Azure.

![screen7](/assets/blogs/sitefinity-open-ai/screen7.png)

### 5. Set API Version

Check the **stable API Version** by clicking the **question mark** tooltip. In this demo, we are using version **2023-05-15**.

![screen8](/assets/blogs/sitefinity-open-ai/screen8.png)

### 6. Save and Restart

Save your changes and **restart** the Sitefinity instance to ensure the configuration is applied correctly. *(Restart is required for proper integration)*.
